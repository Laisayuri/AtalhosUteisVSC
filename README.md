# Atalhos Úteis VSC

###

* executar o arquivo
CTRL + F5

* formatar o documento
ALT + SHIFT + F

* comentários
de linha: CTRL + ;
de bloco: ALT + SHIFT + A

* renomear
F2

* mover linha
ALT + seta

* duplicar linha
ALT + SHIFT + seta

* selecionar próximo igual
CTRL + D

* organizar import's
ALT + SHIFT + O
